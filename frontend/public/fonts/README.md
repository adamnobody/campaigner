Place your font files in this folder and edit `my-local-font.css`.

Quick setup:
1) Copy your `.woff2` files into `frontend/public/fonts/`
2) Update file names in `my-local-font.css`
3) In Appearance settings choose preset: `Локальный шрифт из /public/fonts`
4) Keep font-family name as `"MyLocalFont"` (or change it in both CSS and UI fields)
